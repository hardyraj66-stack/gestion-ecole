# Manuel — Planning

## Accéder au planning

Deux façons d'accéder au planning :
- **Menu Planning** : planning global avec sélecteur de classe
- **Depuis une classe** : bouton "Planning" sur la carte ou la page d'une classe → planning pré-sélectionné pour cette classe

---

## Lire la grille de planning

La grille affiche une semaine type (Lundi → Samedi) avec les heures en colonne gauche.

Chaque créneau est représenté par un bloc coloré (couleur de la matière) affichant :
- Le nom de la matière
- La salle
- Le nom du professeur
- Les horaires

---

## Ajouter un créneau

1. Cliquer sur une cellule vide (icône "+" au survol)
2. Dans la modale qui s'ouvre, remplir :
   - **Matière** (liste des matières de la classe)
   - **Jour** (pré-sélectionné selon la colonne cliquée)
   - **Heure de début**
   - **Heure de fin**
   - **Salle** (salles disponibles sur ce créneau)
   - **Professeur** (optionnel)
3. Cliquer sur **Créer le créneau**

> ⚠️ Si la salle est déjà occupée sur ce créneau par une autre classe, un message d'erreur s'affiche et le créneau n'est pas créé.

---

## Modifier un créneau

1. Cliquer sur un créneau existant
2. Modifier les champs souhaités dans la modale
3. Cliquer sur **Enregistrer**

---

## Déplacer un créneau (glisser-déposer)

1. Cliquer et maintenir sur un créneau
2. Le faire glisser vers la nouvelle cellule horaire
3. Relâcher

Si la destination est occupée (conflit de salle ou de classe), le déplacement est annulé et un message d'erreur s'affiche.

---

## Supprimer un créneau

1. Cliquer sur le créneau
2. Cliquer sur **Supprimer** dans la modale
3. Confirmer

---

## Sélecteur de salle disponible

Lors de la création ou modification d'un créneau, le sélecteur de salle indique pour chaque salle :
- ✅ **Disponible** : la salle est libre sur ce créneau
- ❌ **Occupée** : la salle est utilisée par une autre classe (avec le nom de la classe et la matière affichés)

---

## Types de salle et planning

- **Classe à salle fixe** : la salle est pré-remplie automatiquement dans le créneau
- **Classe à salle variable** : la salle doit être choisie pour chaque créneau
