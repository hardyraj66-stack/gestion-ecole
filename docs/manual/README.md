## Démarrage rapide

1. **Configurer les niveaux** (menu Niveaux) — définir la structure de l'établissement
2. **Créer les matières** (menu Matières) — avec coefficients par niveau
3. **Créer les salles** (menu Salles)
4. **Créer les classes** (menu Classes)
5. **Inscrire les élèves** (menu Élèves)
6. **Configurer le planning** (menu Planning) — pour chaque classe
7. **Saisir les notes** (menu Notes ou Évaluations)
8. **Consulter les bulletins** depuis la fiche de chaque élève
