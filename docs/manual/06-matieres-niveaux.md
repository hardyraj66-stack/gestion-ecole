# Manuel — Matières et Niveaux

## Niveaux scolaires

Les niveaux définissent la structure de l'établissement (ex : 6ème, 5ème, 4ème, 3ème, 2nde, 1ère, Terminale).

### Consulter les niveaux

Aller dans **Niveaux**. Chaque niveau affiche :
- Son nom et son ordre (numéro de tri)
- Les matières qui lui sont associées (pastilles colorées)
- Le nombre de classes du niveau

### Créer un niveau

1. Cliquer sur **Nouveau niveau**
2. Remplir :
   - **Nom** (ex : "6ème")
   - **Ordre** (ex : 1 pour le premier niveau)
   - **Description** (optionnel)
   - **Matières** : sélectionner les matières enseignées dans ce niveau
3. Cliquer sur **Enregistrer**

### Modifier un niveau

Cliquer sur **Modifier** à côté du niveau → une modale s'ouvre avec les champs éditables.

### Supprimer un niveau

Cliquer sur **Supprimer** → confirmation requise.

> ⚠️ Un niveau ne peut pas être supprimé s'il a des classes actives.

## Matières

### Consulter les matières

La page **Matières** affiche toutes les matières avec leur code, couleur et coefficients.

### Filtre par niveau

Sélectionner un niveau dans le filtre pour n'afficher que les matières enseignées à ce niveau.

### Créer une matière

1. Cliquer sur **Nouvelle matière**
2. Remplir :
   - **Nom** (ex : "Mathématiques")
   - **Code** (ex : "MATH" — code court utilisé dans les tableaux)
   - **Description** (optionnel)
   - **Couleur** : cliquer sur une couleur parmi les suggestions ou utiliser le sélecteur
   - **Coefficients par niveau** : pour chaque niveau, indiquer le coefficient de la matière
3. Une prévisualisation de la carte matière s'affiche en temps réel à droite
4. Cliquer sur **Créer la matière**

> 💡 Le coefficient peut différer selon le niveau : les Mathématiques peuvent avoir un coefficient 4 en Terminale et 3 en 2nde.

### Modifier une matière

Depuis la liste des matières → cliquer sur la carte → **Modifier**.
