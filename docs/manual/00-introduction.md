# Manuel d'utilisation — GestionÉcole

## Introduction

**GestionÉcole** est une application de gestion scolaire complète permettant de gérer :

- Les classes et les niveaux scolaires
- Les élèves et leur suivi (assiduité, avertissements, statut)
- Les matières et les coefficients par niveau
- La saisie et le suivi des notes
- Le planning hebdomadaire des cours
- Les salles de l'établissement
- Les professeurs et leurs affectations
- Les évaluations (DS et contrôles)
- Les années scolaires et les archives

## Accès à l'application

Ouvrir un navigateur et accéder à l'URL fournie par votre administrateur (en local : `http://localhost:5173`).

L'application ne nécessite pas de connexion — elle est destinée à un usage en réseau local sécurisé au sein de l'établissement.

## Navigation

La barre latérale gauche donne accès à toutes les sections de l'application :

| Section | Description |
|---------|-------------|
| **Tableau de bord** | Vue synthétique de l'établissement |
| **Classes** | Gestion des classes |
| **Élèves** | Gestion des élèves |
| **Matières** | Gestion des matières |
| **Notes** | Saisie des notes |
| **Planning** | Planning hebdomadaire |
| **Salles** | Gestion des salles |
| **Niveaux** | Niveaux scolaires |
| **Professeurs** | Gestion des professeurs |
| **Évaluations** | Gestion des évaluations |
| **Année scolaire** | Gestion du cycle scolaire |
| **Paramètres** | Personnalisation de l'interface |

## Mise à jour en temps réel

L'application se met à jour automatiquement : si un collègue effectue une modification sur un autre poste, votre écran se rafraîchit sans intervention de votre part.

## Mode archive

Il est possible de consulter les données d'une année scolaire passée en mode lecture seule. Un bandeau orange s'affiche en haut de l'écran pour indiquer ce mode. Toutes les pages fonctionnent normalement, mais aucune modification n'est possible.
