# Manuel — Tableau de bord

## Vue d'ensemble

Le tableau de bord est la page d'accueil de l'application. Il donne une vue synthétique et rapide de l'établissement.

---

## Statistiques globales

Quatre compteurs s'affichent en haut de page :
- **Nombre de classes** actives dans l'année
- **Nombre d'élèves** actifs
- **Nombre de matières**
- **Nombre de salles**

Ces chiffres correspondent toujours à l'année scolaire en cours (ou à l'année consultée en mode archive).

---

## Liste des classes

Un tableau liste les classes avec, pour chacune :
- Nom et niveau
- Nombre d'élèves / Capacité
- Taux de remplissage

Les classes sont paginées (5 par page). Utiliser les boutons Précédent / Suivant pour naviguer.

Cliquer sur une classe → accès direct à la liste des élèves de cette classe.

---

## Convocations en attente

Un encadré liste les convocations des parents qui n'ont pas encore eu lieu. Pour chaque convocation :
- Nom de l'élève
- Date de convocation prévue
- Raison

Permet de suivre rapidement les rendez-vous à venir avec les familles.

---

## Élèves récemment inscrits

Liste les 5 derniers élèves inscrits dans l'établissement avec leur classe.

Cliquer sur un élève → accès à sa fiche.

---

## Actions rapides

Des boutons d'accès rapide permettent d'aller directement aux actions les plus courantes :
- Nouvelle classe
- Nouvel élève
- Saisir des notes
- Voir le planning
