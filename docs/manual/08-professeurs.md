# Manuel — Professeurs

## Consulter la liste des professeurs

La page **Professeurs** affiche tous les professeurs de l'établissement avec leur statut (actif/inactif).

### Rechercher un professeur

Utiliser le champ de recherche pour filtrer par nom ou prénom.

---

## Ajouter un professeur

1. Cliquer sur **Nouveau professeur**
2. Remplir dans la modale :
   - Nom, Prénom
   - Email
   - Téléphone
   - Genre
3. Cliquer sur **Enregistrer**

Le professeur est créé avec le statut **Actif** par défaut.

---

## Fiche professeur

Cliquer sur **Fiche** pour accéder à la page de détail d'un professeur.

La fiche affiche :
- Les informations personnelles
- La liste des **affectations** : classe et matière enseignée
- Le nombre total d'élèves dans ses classes

### Modifier les informations

Cliquer sur **Modifier** → modale d'édition.

### Désactiver un professeur

Cliquer sur **Désactiver** → le professeur passe en statut Inactif et n'apparaît plus dans les sélecteurs du planning.

### Réactiver un professeur

Si le professeur est inactif, un bouton **Activer** est disponible.

---

## Affectations (classes et matières)

La page **Affectations** permet de définir quel professeur enseigne quelle matière dans quelle classe.

1. Aller dans **Professeurs** → **Affectations**
2. La grille affiche toutes les combinaisons classe × matière
3. Pour chaque case, sélectionner le professeur dans la liste déroulante
4. Laisser vide si la matière n'est pas enseignée dans la classe
5. Les modifications sont sauvegardées automatiquement

> 💡 Ces affectations servent à pré-remplir le professeur lors de la création de créneaux de planning.

---

## Professeurs dans le planning

Lors de la création d'un créneau de planning, la liste des professeurs proposés est celle des professeurs actifs affectés à la matière et la classe concernées.
