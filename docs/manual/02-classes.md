# Manuel — Gestion des classes

## Consulter les classes

La page **Classes** affiche toutes les classes de l'année active sous forme de cartes.

Chaque carte affiche :
- Le nom de la classe (ex : "6ème A")
- Le niveau
- La salle assignée (ou "Variable")
- Le nombre d'élèves et la capacité
- Une barre de remplissage (vert = OK, orange = presque plein, rouge = plein)

### Filtrer les classes

- **Recherche** : saisir le nom ou une partie du nom de la classe
- **Filtre par niveau** : sélectionner un niveau dans la liste déroulante

---

## Créer une classe

1. Cliquer sur **Nouvelle classe**
2. Remplir le formulaire :
   - **Nom** (ex : "6ème A")
   - **Niveau** (sélectionner dans la liste)
   - **Capacité** (nombre maximum d'élèves)
   - **Type de salle** :
     - *Salle fixe* : la classe a toujours la même salle → sélectionner la salle
     - *Salle variable* : la salle est choisie pour chaque cours dans le planning
3. Cliquer sur **Créer la classe**

> 💡 L'année scolaire est automatiquement renseignée avec l'année active.

---

## Voir les élèves d'une classe

Depuis la liste des classes, cliquer sur **Voir les élèves** sur la carte d'une classe.

La page affiche :
- Les informations de la classe (niveau, salle, taux de remplissage)
- La liste des élèves avec leur statut
- Un champ de recherche par nom/prénom

Cliquer sur un élève pour accéder à sa fiche complète.

---

## Voir le planning d'une classe

Depuis la liste des classes, cliquer sur **Planning** sur la carte d'une classe.

---

## Désactiver une classe

Une classe peut être désactivée (elle n'apparaît plus dans les listes) mais ses données sont conservées. Cette action est accessible depuis la fiche de la classe.
