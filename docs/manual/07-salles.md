# Manuel — Salles

## Consulter les salles

La page **Salles** affiche toutes les salles de l'établissement sous forme de cartes.

Chaque carte affiche :
- Le nom de la salle
- Le type (Salle de classe, Laboratoire, Salle informatique, etc.)
- La capacité
- Le bâtiment et l'étage
- Les équipements disponibles (projecteur, ordinateurs, tableau interactif, etc.)
- L'accessibilité PMR

### Filtrer les salles

- **Recherche** : par nom de salle
- **Type** : filtrer par type de salle

---

## Créer une salle

1. Cliquer sur **Nouvelle salle**
2. Remplir :
   - **Nom** (ex : "Salle 101")
   - **Capacité**
   - **Type** : Salle de classe, Laboratoire, Salle informatique, Salle de sport, Salle arts, Amphi, Autre
   - **Description** (optionnel)
   - **Bâtiment** (optionnel)
   - **Étage** (optionnel)
   - **Accessible PMR** : cocher si la salle est accessible aux personnes à mobilité réduite
   - **Équipements** : cocher les équipements disponibles
3. Cliquer sur **Enregistrer**

---

## Voir le détail d'une salle

Cliquer sur **Détail** sur la carte d'une salle.

La modale de détail affiche :
- Toutes les informations de la salle
- **Statistiques d'occupation** :
  - Nombre de créneaux par semaine
  - Nombre de jours utilisés
  - Heures d'utilisation par semaine
  - Taux d'occupation (%)
- La liste des créneaux utilisant cette salle

---

## Modifier une salle

Cliquer sur **Modifier** sur la carte → une modale d'édition s'ouvre.

Cliquer sur **Enregistrer** pour valider les modifications.

---

## Supprimer une salle

Dans la modale d'édition, cliquer sur **Supprimer** → confirmation requise.

> ⚠️ La suppression peut être bloquée si des créneaux de planning utilisent cette salle. Retirer d'abord la salle des créneaux concernés.

---

## Disponibilité des salles

La disponibilité en temps réel est consultable dans le **planning** lors de la création d'un créneau. Le sélecteur de salle indique quelles salles sont libres sur le créneau horaire demandé.
