À propos de l'application

GestionÉcole est une application de gestion scolaire complète permettant de gérer :

  • Les classes et les niveaux scolaires
  • Les élèves et leur suivi (assiduité, avertissements, statut)
  • Les matières et les coefficients par niveau
  • La saisie et le suivi des notes
  • Le planning hebdomadaire des cours
  • Les salles de l'établissement
  • Les professeurs et leurs affectations
  • Les évaluations (DS et contrôles)
  • Les années scolaires et les archives

Accès à l'application

Ouvrir un navigateur et accéder à l'URL fournie par votre administrateur (en local : http://localhost:5173). L'application ne nécessite pas de connexion — elle est destinée à un usage en réseau local sécurisé au sein de l'établissement.

L'application se met à jour automatiquement : si un collègue effectue une modification sur un autre poste, votre écran se rafraîchit sans intervention de votre part.

Mode archive

Il est possible de consulter les données d'une année scolaire passée en mode lecture seule. Un bandeau orange s'affiche en haut de l'écran pour indiquer ce mode. Toutes les pages fonctionnent normalement, mais aucune modification n'est possible.

Chapitre 1

Gestion de l'année scolaire

Cycle d'une année scolaire

Chaque année scolaire passe par 3 états :

En préparation → Active → Terminée
  • En préparation : l'année est configurée mais pas encore démarrée. Les classes et matières peuvent être créées.
  • Active : l'année est en cours. Les notes, absences et évaluations sont saisies.
  • Terminée : l'année est clôturée. Les données sont archivées et consultables.

Préparer une nouvelle année

  1. Aller dans Année scolaire
  2. Cliquer sur Préparer une nouvelle année
  3. Renseigner le label (ex. "2025-2026"), la date de début et la date de fin
  4. Cliquer sur Enregistrer
⚠️ Une seule année peut être "en préparation" à la fois.

Démarrer une année

  1. Dans Année scolaire, repérer l'année en préparation
  2. Cliquer sur Démarrer l'année
  3. Confirmer dans la boîte de dialogue

L'année passe en statut Active. Si une autre année était active, elle passe automatiquement en Terminée.

🚨 Cette action est irréversible. Assurez-vous que toutes les classes et matières sont correctement configurées avant de démarrer.

Clôturer une année

  1. Dans Année scolaire, cliquer sur Clôturer l'année
  2. Confirmer dans la boîte de dialogue

L'année passe en statut Terminée. Les données restent accessibles en mode archive.

Consulter une archive

  1. Dans Année scolaire, section Archives
  2. Cliquer sur Consulter pour l'année souhaitée
  3. Un bandeau orange s'affiche : "Mode consultation — Année XXXX-XXXX — Données en lecture seule"
  4. Naviguer normalement dans l'application — tout est consultable
  5. Cliquer sur Quitter dans le bandeau orange pour revenir au mode live

Historique des actions

Chaque action sur une année (démarrage, clôture) est consignée dans l'historique visible sur la page Année scolaire, avec la date et les détails.

Chapitre 2

Gestion des classes

Consulter les classes

La page Classes affiche toutes les classes de l'année active sous forme de cartes. Chaque carte affiche le nom, le niveau, la salle assignée (ou "Variable"), le nombre d'élèves/capacité et une barre de remplissage.

Filtrer les classes

  • Recherche : saisir le nom ou une partie du nom de la classe
  • Filtre par niveau : sélectionner un niveau dans la liste déroulante

Créer une classe

  1. Cliquer sur Nouvelle classe
  2. Remplir le nom (ex : "6ème A"), le niveau, la capacité
  3. Choisir le type de salle :
    • Salle fixe : la classe a toujours la même salle → sélectionner la salle
    • Salle variable : la salle est choisie pour chaque cours dans le planning
  4. Cliquer sur Créer la classe
💡 L'année scolaire est automatiquement renseignée avec l'année active.

Voir les élèves d'une classe

Depuis la liste des classes, cliquer sur Voir les élèves sur la carte d'une classe. La page affiche les informations de la classe, la liste des élèves avec leur statut, et un champ de recherche par nom/prénom.

Voir le planning d'une classe

Depuis la liste des classes, cliquer sur Planning sur la carte d'une classe.

Désactiver une classe

Une classe peut être désactivée (elle n'apparaît plus dans les listes) mais ses données sont conservées. Cette action est accessible depuis la fiche de la classe.

Chapitre 3

Gestion des élèves

Consulter la liste des élèves

La page Élèves affiche tous les élèves de l'établissement. Filtres disponibles : recherche par nom/prénom, filtre par classe.

Statuts des élèves

StatutSignification
ActifÉlève en cours de scolarité
ExcluÉlève exclu de l'établissement
PartiÉlève ayant quitté l'établissement

Inscrire un nouvel élève

  1. Cliquer sur Nouvel élève
  2. Remplir la section Identité : Nom, Prénom, Date de naissance, Genre, Classe d'affectation, Email/Téléphone/Adresse (optionnels)
  3. Remplir optionnellement la section Famille : père, mère, tuteur
  4. Cliquer sur Enregistrer l'élève

Fiche élève — 5 onglets

Cliquer sur un élève pour accéder à sa fiche complète :

Onglet Identité

Informations personnelles. Bouton Modifier pour mettre à jour.

Onglet Famille

Informations des parents/tuteur.

Onglet Assiduité

Tableau de toutes les absences et retards. Bouton Ajouter une absence : choisir type (absence/retard), date, durée, motif, justifiée ou non. Compteurs : total absences, retards, justifiées.

Onglet Avertissements

Liste des avertissements et des convocations des parents.

⚠️ Après 3 avertissements, une convocation est automatiquement générée.

Onglet Statut

Gestion du statut de l'élève :

  • Exclure : renseigner la raison → l'élève passe en statut Exclu
  • Élève parti : choisir motif, date de départ → statut Parti
  • Réintégrer : l'élève repasse en statut Actif

Bulletin de notes

Depuis la fiche élève → Bulletin. Affiche pour le trimestre sélectionné : matières avec DS/Évaluation/Moyenne/Coefficient, et la moyenne générale pondérée.

Codes couleur des notes : Rouge < 8 · Orange 8–10 · Jaune 10–14 · Vert ≥ 14

Chapitre 4

Notes et Évaluations

Deux façons de saisir des notes

1. Saisie directe (page Notes)

Pour saisir des notes rapidement sans créer d'évaluation formelle.

  1. Aller dans Notes
  2. Sélectionner la classe, la matière, le trimestre et le type (DS ou Évaluation)
  3. La liste des élèves de la classe s'affiche
  4. Pour chaque élève : saisir la note (0–20)
  5. Cliquer sur Enregistrer
💡 Si une note existe déjà pour cet élève/matière/trimestre/type, elle est affichée et peut être modifiée.

Annuler une note

Cliquer sur Annuler à côté d'une note. Elle est marquée comme annulée (ne compte plus dans les calculs) mais reste dans la base de données.

2. Via les évaluations (page Évaluations)

Plus structuré : permet de créer une évaluation formelle, saisir toutes les notes, puis la publier.

Créer une évaluation

  1. Aller dans Évaluations → Liste → Nouvelle évaluation
  2. Remplir le type (DS ou Évaluation), la classe, la matière, le trimestre, la date
  3. Cliquer sur Créer → redirigé vers la page de saisie

Saisir les notes d'une évaluation

Sur la page de détail (statut : Brouillon) : pour chaque élève, saisir la note ou cocher Absent. Cliquer sur Enregistrer les notes.

💡 Un élève absent ne reçoit pas de note — il n'est pas pénalisé dans le calcul du bulletin.

Publier une évaluation

  1. Quand toutes les notes sont saisies, cliquer sur Publier
  2. Confirmer

La publication : verrouille l'évaluation, crée automatiquement les notes dans la base de données, affiche les statistiques finales.

🚨 Une évaluation publiée ne peut pas être supprimée ni modifiée.

Périodes d'évaluation

Évaluations → Périodes — configure les dates des périodes DS et Évaluation pour chaque trimestre. Cliquer sur les champs de dates pour les modifier. Bouton Terminer pour clôturer une période.

Consulter le bulletin d'un élève

Depuis la fiche d'un élève → Bulletin. Affiche pour chaque trimestre toutes les matières avec DS, Évaluation, Moyenne, Coefficient, la moyenne générale pondérée et la mention.

Chapitre 5

Planning

Accéder au planning

Deux façons d'accéder au planning : menu Planning (planning global avec sélecteur de classe), ou depuis une classe via le bouton "Planning" sur la carte.

Lire la grille de planning

La grille affiche une semaine type (Lundi → Samedi) avec les heures en colonne gauche. Chaque créneau est représenté par un bloc coloré (couleur de la matière) affichant le nom de la matière, la salle, le professeur et les horaires.

Ajouter un créneau

  1. Cliquer sur une cellule vide (icône "+" au survol)
  2. Remplir la matière, le jour, l'heure de début, l'heure de fin, la salle
  3. Cliquer sur Créer le créneau
⚠️ Si la salle est déjà occupée sur ce créneau par une autre classe, un message d'erreur s'affiche.

Modifier un créneau

  1. Cliquer sur un créneau existant
  2. Modifier les champs souhaités dans la modale
  3. Cliquer sur Enregistrer

Déplacer un créneau (glisser-déposer)

  1. Cliquer et maintenir sur un créneau
  2. Le faire glisser vers la nouvelle cellule horaire
  3. Relâcher

Si la destination est occupée (conflit de salle ou de classe), le déplacement est annulé.

Supprimer un créneau

  1. Cliquer sur le créneau
  2. Cliquer sur Supprimer dans la modale
  3. Confirmer

Sélecteur de salle disponible

Lors de la création ou modification d'un créneau, le sélecteur de salle indique pour chaque salle :

  • ✅ Disponible : la salle est libre sur ce créneau
  • ❌ Occupée : utilisée par une autre classe (avec le nom de la classe affiché)
Chapitre 6

Matières et Niveaux

Niveaux scolaires

Les niveaux définissent la structure de l'établissement (ex : 6ème, 5ème, 4ème, 3ème, 2nde, 1ère, Terminale).

Consulter les niveaux

Aller dans Niveaux. Chaque niveau affiche son nom, son ordre, les matières associées et le nombre de classes.

Créer un niveau

  1. Cliquer sur Nouveau niveau
  2. Remplir le nom (ex : "6ème"), l'ordre (ex : 1), une description (optionnel) et les matières enseignées
  3. Cliquer sur Enregistrer

Modifier ou supprimer un niveau

Cliquer sur Modifier à côté du niveau. Pour supprimer, utiliser le bouton Supprimer.

⚠️ Un niveau ne peut pas être supprimé s'il a des classes actives.

Matières

La page Matières affiche toutes les matières avec leur code, couleur et coefficients.

Créer une matière

  1. Cliquer sur Nouvelle matière
  2. Remplir le nom (ex : "Mathématiques"), le code (ex : "MATH"), la description (optionnel), la couleur
  3. Renseigner les coefficients par niveau : pour chaque niveau, indiquer le coefficient de la matière
  4. Une prévisualisation de la carte s'affiche en temps réel
  5. Cliquer sur Créer la matière
💡 Le coefficient peut différer selon le niveau : les Mathématiques peuvent avoir un coefficient 4 en Terminale et 3 en 2nde.

Modifier une matière

Depuis la liste des matières → cliquer sur la carte → Modifier.

Chapitre 7

Salles

Consulter les salles

La page Salles affiche toutes les salles de l'établissement sous forme de cartes : nom, type, capacité, bâtiment, étage, équipements, accessibilité PMR.

Filtrer les salles

  • Recherche : par nom de salle
  • Type : filtrer par type de salle

Créer une salle

  1. Cliquer sur Nouvelle salle
  2. Remplir le nom, la capacité, le type, le bâtiment et l'étage (optionnels)
  3. Cocher les équipements disponibles et l'accessibilité PMR
  4. Cliquer sur Enregistrer

Types de salle

TypeUsage
Salle de classeCours standard
LaboratoireSciences expérimentales
Salle informatiqueCours d'informatique
Salle de sportÉducation physique
Salle artsArts plastiques / musique
AmphiAmphithéâtre
AutreUsage divers

Voir le détail d'une salle

Cliquer sur Détail sur la carte d'une salle. Affiche : toutes les informations, statistiques d'occupation (créneaux/semaine, taux d'occupation), et la liste des créneaux utilisant cette salle.

Modifier et supprimer une salle

Cliquer sur Modifier sur la carte. Pour supprimer, utiliser le bouton Supprimer dans la modale d'édition.

⚠️ La suppression peut être bloquée si des créneaux de planning utilisent cette salle. Retirer d'abord la salle des créneaux concernés.

Disponibilité des salles

La disponibilité en temps réel est consultable dans le planning lors de la création d'un créneau. Le sélecteur indique quelles salles sont libres sur le créneau demandé.

Chapitre 8

Professeurs

Consulter la liste des professeurs

La page Professeurs affiche tous les professeurs de l'établissement avec leur statut (Actif / Inactif). Utiliser le champ de recherche pour filtrer par nom ou prénom.

Ajouter un professeur

  1. Cliquer sur Nouveau professeur
  2. Remplir : Nom, Prénom, Email, Téléphone, Genre
  3. Cliquer sur Enregistrer

Le professeur est créé avec le statut Actif par défaut.

Fiche professeur

Cliquer sur Fiche pour accéder à la page de détail. Affiche les informations personnelles et la liste des affectations (classe et matière enseignée).

  • Modifier les informations → modale d'édition
  • Désactiver → passe en statut Inactif, n'apparaît plus dans les sélecteurs du planning
  • Activer → bouton disponible si le professeur est inactif

Affectations (classes et matières)

La page Affectations permet de définir quel professeur enseigne quelle matière dans quelle classe.

  1. Aller dans Professeurs → Affectations
  2. La grille affiche toutes les combinaisons classe × matière
  3. Pour chaque case, sélectionner le professeur dans la liste déroulante
  4. Les modifications sont sauvegardées automatiquement
💡 Ces affectations servent à pré-remplir le professeur lors de la création de créneaux de planning.
Chapitre 9

Paramètres et personnalisation

Accéder aux paramètres

Cliquer sur Paramètres en bas de la barre latérale.

Thème sombre / clair

Dans la section Apparence :

  • Clair : interface avec fond blanc (défaut)
  • Sombre : interface avec fond sombre, idéale pour une utilisation en faible luminosité

Le changement est immédiat et persisté entre les sessions.

Couleur principale

Choisir parmi 6 couleurs prédéfinies :

  • 🔵 Bleu (défaut)
  • 🟣 Violet
  • 🟢 Vert
  • 🟡 Orange
  • 🩷 Rose
  • 🔷 Indigo

La couleur est appliquée aux boutons, liens actifs, barres de progression et éléments interactifs.

Langue d'interface

Trois langues sont disponibles :

  • Français (défaut)
  • English
  • Malagasy

Le changement est immédiat — toute l'interface est traduite.

Persistance des préférences

Les préférences sont enregistrées dans le navigateur (IndexedDB). Elles sont conservées même après fermeture du navigateur, mais sont spécifiques à l'appareil et au navigateur utilisés.

Chapitre 10

Tableau de bord

Vue d'ensemble

Le tableau de bord est la page d'accueil de l'application. Il donne une vue synthétique et rapide de l'établissement.

Statistiques globales

Quatre compteurs s'affichent en haut de page :

  • Nombre de classes actives dans l'année
  • Nombre d'élèves actifs
  • Nombre de matières
  • Nombre de salles

Ces chiffres correspondent toujours à l'année scolaire en cours (ou à l'année consultée en mode archive).

Liste des classes

Un tableau liste les classes avec, pour chacune : nom/niveau, nombre d'élèves/capacité, taux de remplissage. Les classes sont paginées (5 par page). Cliquer sur une classe → accès direct à la liste des élèves.

Convocations en attente

Un encadré liste les convocations des parents qui n'ont pas encore eu lieu : nom de l'élève, date de convocation prévue, raison. Permet de suivre rapidement les rendez-vous à venir avec les familles.

Élèves récemment inscrits

Liste les 5 derniers élèves inscrits dans l'établissement avec leur classe. Cliquer sur un élève → accès à sa fiche.

Actions rapides

Des boutons d'accès rapide permettent d'aller directement aux actions les plus courantes :

  • Nouvelle classe
  • Nouvel élève
  • Saisir des notes
  • Voir le planning
