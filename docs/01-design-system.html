🎨 Design System

Variables CSS, couleurs, typographie, layout, dark mode, thèmes de couleur

Le design system est implémenté en SCSS dans frontend/src/styles/ : _variables.scss (variables Sass), _themes.scss (CSS custom properties — clair par défaut, sombre, couleurs d'accent), puis des partials components/, layout/ et pages/. Les classes utilitaires Tailwind v4 restent disponibles en complément (cn() = clsx + tailwind-merge). La marque Ekolova est rendue par les composants components/brand/.

Dark mode

Activé via l'attribut data-theme="dark" sur <html>, positionné par SettingsContext au démarrage. Toutes les variables ci-dessous sont surchargées automatiquement — aucune classe CSS à ajouter dans les composants.

VariableLightDark
--bg#f1f5f9#0f172a
--card-bg#ffffff#1e293b
--text#0f172a#f1f5f9
--text-muted#64748b#94a3b8
--border#e2e8f0#334155
--primary-light#eff6ff#1e3a5f (et variantes par couleur)

Le toggle dark/light est dans la page Paramètres (/parametres). Le choix est persisté dans IndexedDB et appliqué avant le premier rendu pour éviter le flash de mauvais thème.
