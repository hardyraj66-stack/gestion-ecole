📋 Module Évaluations & Périodes

Gestion des évaluations (DS, évaluations libres) et des fenêtres de saisie par trimestre

Vue d'ensemble

Deux entités distinctes mais liées :

  • Évaluation — Un devoir ou une évaluation libre pour une classe, une matière et un trimestre. Contient les notes de chaque élève.
  • PériodeEvaluation — Une fenêtre calendaire (date_debut → date_fin) qui contrôle la période de saisie d'un type d'évaluation pour un trimestre donné.

API Évaluations

POST /evaluations

Crée une évaluation. Émet evaluation:created.

{
  "type": "ds",
  "classe_id": "...",
  "matiere_id": "...",
  "trimestre": 1,
  "annee_scolaire": "2024-2025",
  "date": "2025-01-15"
}

PATCH /evaluations/:id/notes

Saisie ou mise à jour des notes des élèves. Émet evaluation:updated.

{
  "notes": [
    { "eleve_id": "...", "valeur": 14.5, "absent": false },
    { "eleve_id": "...", "valeur": null, "absent": true }
  ]
}

PATCH /evaluations/:id/publier

Passe le statut de 'brouillon' à 'publie'. Une évaluation publiée ne peut plus être modifiée. Émet evaluation:publie.

DELETE /evaluations/:id

Supprime une évaluation (brouillon uniquement recommandé). Émet evaluation:deleted.

Schéma PériodeEvaluation

Index unique : (trimestre, type, annee_scolaire) — 6 périodes max par année scolaire (3 trimestres × 2 types).

API Périodes

GET /periodes?annee_scolaire=2024-2025

Retourne les 6 périodes de l'année scolaire. Crée celles manquantes automatiquement.

GET /periodes/active

Retourne la période en cours (date_debut ≤ aujourd'hui ≤ date_fin et non terminée).

POST /periodes/init

Initialise les 6 périodes pour une année scolaire si elles n'existent pas encore.

{ "annee_scolaire": "2024-2025" }

PATCH /periodes/:id

Met à jour les dates d'une période. Émet periode:updated.

{
  "date_debut": "2025-01-06",
  "date_fin": "2025-01-20"
}

PATCH /periodes/:id/terminer

Marque la période comme terminée (terminee: true). Émet periode:updated.

Page frontend — PériodesList

Route : /evaluations
Fichier : src/pages/evaluations/PeriodesList.tsx

Affiche une grille des 6 périodes (2 types × 3 trimestres) avec :

  • Dates de début et de fin éditables inline
  • Statut visuel : à planifier, en cours, terminée
  • Bouton Terminer pour clore une période
  • Accès rapide aux évaluations de chaque période
