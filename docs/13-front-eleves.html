👥 Gestion des Élèves

Pages /eleves, /eleves/nouveau, /eleves/:id/bulletin

Liste — /eleves

  • FilterBar : champ recherche (nom/prénom) + select classe + compteur
  • Table : paginée (12/page), colonnes avatar, genre, classe (lien), email, actions
  • Pagination reset automatique quand les filtres changent (via key)
  • Utilise useData() pour supporter le mode archive

Inscription — /eleves/nouveau

Formulaire en 3 sections :

  1. Informations personnelles (3 colonnes) : prénom*, nom*, genre*
  2. Scolarité (2 colonnes) : date de naissance*, classe*
  3. Contact (2 colonnes) : email, téléphone, adresse

Bulletin — /eleves/:id/bulletin

  • Layout 2 colonnes : StudentCard (avatar, nom, classe, moyenne générale) + contenu
  • TrimestreTabs : 3 onglets cliquables
  • GradesTable : matière, code, coefficient, notes (chips colorées), moyenne, mention
  • Pied de tableau avec la moyenne générale pondérée
  • Le bulletin est chargé depuis GET /notes/bulletin/:id?trimestre=X à chaque changement de trimestre
