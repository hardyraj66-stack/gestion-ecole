🏫 Module Classes

Gestion des classes de l'établissement

Schéma MongoDB

{
  nom:            String   (requis)     // "6ème A"
  niveau:         String   (requis)     // "6ème"
  annee_scolaire: String   (requis)     // "2024-2025"
  capacite:       Number   (défaut: 30) // places max
  salle:          String   (requis)     // "Salle 101"
  salle_type:     Enum     ['fixe','variable']  // mode de salle
  createdAt, updatedAt                  // timestamps auto
}

Endpoints API

Mode de salle

  • fixe : la classe est toujours dans la même salle. Dans le planning, le champ salle est en lecture seule.
  • variable : la salle change selon l'emploi du temps. Le formulaire de créneau affiche un sélecteur avec vérification de disponibilité en temps réel.

Niveaux disponibles

6ème, 5ème, 4ème, 3ème, 2nde, 1ère, Terminale, CP, CE1, CE2, CM1, CM2
