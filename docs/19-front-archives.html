📂 Mode Archive

Consultation des données d'une année scolaire terminée

Déclenchement

L'utilisateur clique « 📂 Consulter » sur une année terminée dans la page Cycle scolaire. Le système :

  1. Charge un snapshot via GET /annees/:id/snapshot
  2. Active le mode archive dans ViewingContext
  3. Redirige vers /dashboard

Architecture technique

ViewingContext

interface ViewingContextType {
  viewing: AnneeScolaire | null;   // année consultée
  snapshot: Snapshot | null;        // données chargées
  loadingSnapshot: boolean;
  viewAnnee(annee): Promise;        // active le mode
  exitView(): void;                 // désactive
  isViewingArchive: boolean;
}

Hook useData()

Remplace les appels directs aux contextes dans les pages :

const { classes, eleves, matieres, notes, creneaux, loading, readOnly } = useData();

// Si isViewingArchive → retourne les données du snapshot + readOnly = true
// Sinon → retourne les données live des contextes + readOnly = false

Comportement des pages en mode archive

PageMode normalMode archive (lecture seule)
DashboardStats live, accès rapidesStats snapshot, accès rapides masqués
ClassesGrille + bouton créer + ✕ supprimerGrille consultation, boutons masqués, carte « + » masquée
Classe → ÉlèvesTableau + bouton nouvel élève + ✕ supprimerTableau lecture seule, tous boutons de mutation masqués
ÉlèvesTableau filtré + bouton inscrire + ✕ supprimerTableau lecture seule, boutons masqués
MatièresGrille + bouton créer + supprimerGrille consultation, boutons masqués
SallesGrille + formulaire inline + modifier/supprimerGrille lecture seule, formulaire bloqué, alerte archive
NotesSaisie en lot, enregistrementConsultation uniquement, inputs désactivés, alerte archive
PlanningCalendrier + formulaire créneau + ✕ supprimerCalendrier lecture seule, formulaire bloqué, ✕ masqué
BulletinBulletin + lien « Saisir des notes »Bulletin lecture seule, lien saisie masqué
Créer classe/élève/matièreFormulaires fonctionnelsRedirection automatique vers la liste

Sécurité : les pages de création (/classes/nouvelle, /eleves/nouveau, /matieres/nouvelle) détectent le mode archive et redirigent immédiatement avec <Navigate />. Aucun formulaire de mutation n'est accessible en mode archive.

Sortir du mode archive

3 façons de revenir à l'année en cours :

  • Cliquer sur la bannière orange en haut du contenu
  • Cliquer sur l'encart année dans le footer de la sidebar
  • Toute navigation vers la page /annee-scolaire ne sort pas automatiquement du mode (l'utilisateur doit cliquer explicitement)
