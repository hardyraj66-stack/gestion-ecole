📝 Saisie des Notes

Page /notes

Flux utilisateur

  1. Sélectionner une classe, une matière, un trimestre et une date d'évaluation
  2. Cliquer « Charger les élèves »
  3. Saisir les notes (0-20, pas 0.5) et commentaires optionnels pour chaque élève
  4. Cliquer « ✓ Enregistrer toutes les notes »

Composants

ComposantRôle
NotesFiltersCard de sélection (classe, matière, trimestre, date) + bouton charger
NotesStatsBarBande : « X / Y saisies » + « Z — Moy. classe »
NotesTableTableau paginé (10/page) : avatar, nom, input note, input commentaire, badge statut

Logique

  • Au chargement, les notes existantes (même élève + matière + trimestre) sont pré-remplies avec existingId
  • À l'enregistrement : si existingId → PATCH, sinon → POST
  • L'input note a une classe CSS note-good (vert, ≥10) ou note-bad (rouge, <10)
  • Badge statut : « Existante » (gris), « Nouvelle » (vert), ou « — »
