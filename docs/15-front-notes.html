📝 Saisie des Notes

Page /notes

Source des données

La page charge sa configuration via GET /read/notes :

  • classes
  • matières
  • élèves
  • notes existantes

La page de saisie ne pagine plus côté front. Le backend fournit les données et le frontend les regroupe pour la saisie.

Composants

ComposantRôle
NotesFiltersCard de sélection (classe, matière, trimestre, date)
NotesStatsBarBande : « X / Y saisies » + moyenne
NotesTableTableau de saisie avec composants stylés pour note et commentaire

Inputs stylés

  • note-field : champ numérique stylé, couleurs par valeur, indicateur visuel
  • comment-field : champ texte uniforme avec focus et mode lecture seule

Lecture seule archive

En mode archive :

  • les champs sont disabled
  • le bouton d'enregistrement est masqué
  • une alerte orange indique le mode consultation
