Persistance — settingsDB.ts

Base IndexedDB : ekolova-settings v1, store settings, clé 'user'.

interface AppSettings {
  theme:    'light' | 'dark';
  color:    'blue' | 'violet' | 'green' | 'red' | 'orange' | 'indigo';
  language: 'fr' | 'en' | 'mg';
}

// Défauts
{ theme: 'light', color: 'blue', language: 'fr' }
FonctionDescription
loadSettings()Lit depuis IndexedDB, fusionne avec les défauts. Retourne AppSettings.
saveSettings(partial)Fusionne et persiste. Silently fail — les préférences ne sont pas critiques.

SettingsProvider

Doit être le provider le plus externe dans AppProviders.tsx — il bloque le rendu de l'app avec un spinner pendant le chargement des préférences pour éviter un flash de mauvais thème.

// Ordre dans AppProviders.tsx :
<SettingsProvider>          ← 1er — lit IndexedDB, applique thème/couleur/langue
  <ConfirmProvider>
    <AnneeProvider>
      ...
    </AnneeProvider>
  </ConfirmProvider>
</SettingsProvider>

Application au DOM

// Thème → attribut HTML
document.documentElement.setAttribute('data-theme', 'dark');

// Couleur → attribut HTML
document.documentElement.setAttribute('data-color', 'violet');

// Langue → i18next
i18n.changeLanguage('mg');

Interface du contexte

interface SettingsContextType {
  settings: AppSettings;
  ready: boolean;
  setTheme(theme: 'light' | 'dark'): void;
  setColor(color: AppSettings['color']): void;
  setLanguage(language: AppSettings['language']): void;
}

Internationalisation — i18next

Bibliothèques : i18next + react-i18next.

// src/i18n/index.ts
i18n.use(initReactI18next).init({
  resources: { fr: { translation: fr }, en: { translation: en }, mg: { translation: mg } },
  lng: 'fr',
  fallbackLng: 'fr',
  interpolation: { escapeValue: false },
});

./i18n/index est importé dans main.tsx avant App pour garantir que i18next est initialisé avant le premier rendu.

Page Paramètres

Route : /parametres — Fichier : src/pages/Parametres/Parametres.tsx

Structure de la page :

  • Section Apparence
    • Toggle dark/light (.toggle-switch) — changement instantané
    • 6 swatches de couleur (.color-swatches + .color-swatch) — label dessous
  • Section Langue
    • 3 boutons langue (.lang-buttons + .lang-btn) avec drapeau + label
    • Changement instantané via i18n.changeLanguage()
