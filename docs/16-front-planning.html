Sélection de classe

Liste à gauche avec toutes les classes. Chaque item affiche le nom, le niveau, la salle et un badge avec le nombre de créneaux. La classe sélectionnée est surlignée.

Configuration

Barre d'informations : nom de classe, niveau, salle, mode (badge Fixe/Variable), nombre de créneaux, heures/semaine.

Formulaire de créneau

  • Grille 3 colonnes : matière*, jour*, enseignant
  • Grille 3 colonnes : heure début*, heure fin*, salle*
  • Salle en mode fixe : input readonly avec badge « Fixe »
  • Salle en mode variable : select avec vérification de disponibilité en temps réel (re-fetch quand jour/heure changent), légende vert/rouge

Calendrier hebdomadaire

Table planning-table : lignes = heures (07:00→18:00, pas 30min), colonnes = jours (Lun→Sam). Chaque créneau = bloc coloré (couleur matière) avec nom, horaires, salle, enseignant. Bouton ✕ au survol pour supprimer.
