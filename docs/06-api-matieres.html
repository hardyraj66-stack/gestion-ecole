📚 Module Matières

Programme de matières avec coefficients et couleurs

Schéma MongoDB

{
  nom:         String  (requis)           // "Mathématiques"
  code:        String  (requis)           // "MATH" (max 6 car.)
  coefficient: Number  (requis, défaut 1) // 1 à 10
  description: String  (optionnel)
  couleur:     String  (optionnel)        // "#2563eb"
}

Endpoints API

MéthodeRouteDescription
GET/matieresToutes les matières
GET/matieres/:idUne matière
POST/matieresCréer
PATCH/matieres/:idModifier
DELETE/matieres/:idSupprimer
