ConfirmDialog — Détails

Remplace tous les window.confirm() natifs par une popup stylisée.

const confirm = useConfirm();

const ok = await confirm({
  title: 'Supprimer la classe',
  message: 'Êtes-vous sûr ?',
  confirmText: 'Supprimer',
  cancelText: 'Annuler',       // optionnel, défaut "Annuler"
  variant: 'danger',           // danger | warning | info
});

if (ok) { /* supprimer */ }

Architecture : ConfirmProvider est placé tout en haut de l'arbre dans AppProviders. Il gère un état unique (une seule popup à la fois) avec une Promise pour le retour.

Pagination — Détails

import { Pagination, paginate } from './Pagination';

const PAGE_SIZE = 10;
const [page, setPage] = useState(1);
const paged = paginate(items, page, PAGE_SIZE);

// Dans le JSX :
<Pagination
  currentPage={page}
  totalItems={items.length}
  pageSize={PAGE_SIZE}
  onPageChange={setPage}
/>

La pagination affiche des ellipses intelligentes quand il y a plus de 7 pages. Elle se masque automatiquement pour 1 seule page.

SearchInputSuggestions — Détails

Version enrichie de SearchInput avec auto-complétion dynamique et navigation clavier.

const fetchSuggestions = useCallback(async (query: string): Promise<Suggestion[]> => {
  const data = await readApi.elevesList(1, 8, query, '');
  return (data?.eleves ?? []).map(e => ({
    id: e.id,
    label: `${e.prenom} ${e.nom}`,
    sublabel: e.classe_nom,
  }));
}, []);

<SearchInputSuggestions
  placeholder="Rechercher par nom ou prénom…"
  value={search}
  onChange={setSearch}
  onSelect={s => setSearch(s.label)}
  fetchSuggestions={fetchSuggestions}
  debounceMs={250}
/>
  • Suggestions à partir de 2 caractères, debounce configurable
  • Navigation : ↑ / ↓, Entrée pour sélectionner, Échap pour fermer
  • Bouton ✕ pour effacer la saisie
  • Fermeture automatique au clic extérieur

Popover — Détails

Conteneur positionné relatif au trigger, avec gestion du clic extérieur et de la touche Échap.

const [open, setOpen] = useState(false);

<Popover
  open={open}
  onClose={() => setOpen(false)}
  align="left"
  trigger={
    <button onClick={() => setOpen(true)}>Ouvrir</button>
  }
>
  <div>Contenu du popover</div>
</Popover>

Utilisé notamment dans ElevesFiltersBar et NotesFilters pour la sélection niveau→classe en deux étapes.

Button — Détails

Le composant Button peut rendre un <button> ou un <Link> React Router :

// Bouton standard
<Button variant="primary" onClick={save} loading={saving}>
  Sauvegarder
</Button>

// Bouton-lien
<Button as="link" to="/classes/nouvelle" variant="outline">
  + Nouvelle classe
</Button>

ExportMenu — Détails

Bouton dropdown btn-outline btn-sm qui propose le téléchargement CSV et Excel depuis n'importe quelle page liste.

import { ExportMenu } from '../../components/shared/ExportMenu';

// Dans le PageHeader d'une page liste :
<ExportMenu
  csvUrl="/export/eleves/csv?classeId=xxx"
  xlsxUrl="/export/eleves/xlsx?classeId=xxx"
  label="Exporter"  {/* optionnel, défaut "Exporter" */}
/>
  • Dropdown fermé automatiquement au clic extérieur (mousedown listener)
  • Téléchargement via window.open(API_BASE_URL + path, '_blank')
  • Les URLs sont des chemins relatifs — API_BASE_URL est préfixé à l'exécution
  • Classe CSS du menu : .export-menu-item dans index.css

Pour le bulletin PDF et la carte scolaire (pages HTML imprimables), utiliser directement window.open(API_BASE_URL + '/export/bulletin/:id?trimestre=1', '_blank') — pas via ExportMenu.
