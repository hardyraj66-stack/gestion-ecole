📤 Module Export

CSV, Excel (XLS) et documents HTML imprimables (bulletin PDF, carte scolaire)

Aucune dépendance npm externe. Le CSV est généré en pur JS, le XLSX est un fichier XML SpreadsheetML, le bulletin et la carte sont des pages HTML imprimables via window.print().

Architecture

server/src/export/ ├── export.controller.ts # Routes GET /export/* ├── export.service.ts # Logique de génération (CSV, XLSX, HTML) └── export.module.ts src/components/shared/ └── ExportMenu.tsx # Bouton dropdown CSV / Excel dans le header des pages

Documents imprimables

MéthodeRouteParamsDescription
GET/export/bulletin/:eleveId?trimestre=1trimestre (1-3)Page HTML bulletin A4, imprimable directement
GET/export/carte/:eleveId—Page HTML carte scolaire recto/verso (340×210 px)

Le bulletin et la carte retournent text/html, pas un PDF. Le navigateur doit utiliser Imprimer → Enregistrer en PDF. Le bouton Imprimer est intégré à la page HTML générée.

Composant frontend — ExportMenu

Fichier : src/components/shared/ExportMenu.tsx

interface ExportMenuProps {
  csvUrl: string;   // chemin relatif, ex: '/export/eleves/csv?classeId=xxx'
  xlsxUrl: string;  // chemin relatif, ex: '/export/eleves/xlsx?classeId=xxx'
  label?: string;   // défaut: 'Exporter'
}
  • Bouton btn-outline btn-sm avec flèche déroulante
  • Dropdown fermé au clic extérieur (via mousedown listener)
  • Téléchargement via window.open(API_BASE_URL + path, '_blank')
  • Classe CSS du dropdown : .export-menu-item dans index.css

Exemple d'usage dans une page

// Dans ClassesList, ElevesList, MatieresList, etc.
<ExportMenu

  xlsxUrl={`/export/eleves/xlsx${classeId ? `?classeId=${classeId}` : ''}`}

// Bulletin — ouvrir dans un nouvel onglet
window.open(`${API_BASE_URL}/export/bulletin/${eleveId}?trimestre=${trimestre}`, '_blank');

// Carte scolaire
window.open(`${API_BASE_URL}/export/carte/${eleveId}`, '_blank');

Bulletin HTML — structure

La page retournée est un A4 imprimable avec :

  • En-tête école + titre + année scolaire
  • Carte élève (nom, classe, niveau, date de naissance)
  • Badge trimestre
  • Tableau des notes : matière, code, coef., DS, évaluation, moyenne, mention
  • Ligne de pied : moyenne générale pondérée + mention globale
  • Zone de signatures (directeur, professeur principal, parent)
  • Barre no-print avec bouton Imprimer / PDF

Mentions : Très bien ≥16, Bien ≥14, Assez bien ≥12, Passable ≥10, Insuffisant <10.

Carte scolaire — structure

Deux cartes 340×210 px (format CB) affichées l'une sous l'autre :

  • Recto : fond dégradé bleu — avatar initiales coloré (bleu garçon, rose fille) — nom, prénom, classe, niveau — ID tronqué + année
  • Verso : fond blanc — champs classe, niveau, email, téléphone, adresse — date d'émission
