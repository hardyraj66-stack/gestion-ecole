👥 Module Élèves

Gestion des élèves inscrits dans l'établissement

Schéma MongoDB

{
  nom:            String   (requis)
  prenom:         String   (requis)
  date_naissance: String   (requis)   // "2013-05-14"
  genre:          Enum     ['M','F']
  classe_id:      String   (requis)   // référence Classe._id
  email:          String   (optionnel)
  telephone:      String   (optionnel)
  adresse:        String   (optionnel)
}

Endpoints API

MéthodeRouteDescription
GET/elevesTous les élèves
GET/eleves/:idUn élève
POST/elevesInscrire un élève
PATCH/eleves/:idModifier
DELETE/eleves/:idSupprimer

Événements Socket.IO

ÉvénementPayload
eleve:createdObjet Eleve
eleve:updatedObjet Eleve
eleve:deleted{ id }

Lien avec les classes

Le champ classe_id fait le lien avec la collection classes. Le frontend filtre les élèves par classe pour afficher la liste dans la page /classes/:id/eleves.

Helpers frontend

  • getInitials(eleve) → premières lettres de prénom et nom en majuscules
  • getAge(date_naissance) → âge en années entières
  • formatDate(dateStr) → format dd/MM/yyyy
