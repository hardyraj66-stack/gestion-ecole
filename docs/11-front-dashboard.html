📊 Dashboard

Route /dashboard — Vue d'ensemble de l'établissement

Source des données

La page utilise useDashboardData() qui fait un seul appel API vers GET /read/dashboard. Le backend retourne en une fois :

  • stats — classes, élèves, matières, notes
  • classesWithCount — les classes de la page courante du tableau
  • classesPagination — meta de pagination
  • recentEleves — 5 derniers élèves
  • anneeActive

Composants

ZoneFichierDescription
PageDashboard.tsxOrchestre les données via useDashboardData()
Statistiques4 × StatCardClasses, Élèves, Matières, Notes saisies
TableauClassesTable.tsxPagination côté serveur : le changement de page rappelle /read/dashboard?classesPage=X
Derniers élèvesRecentEleves.tsx5 derniers inscrits avec avatar, classe, lien bulletin
Accès rapidesQuickActions.tsx4 liens : nouvelle classe, nouvel élève, notes, planning

Intégration année scolaire

  • En-tête : badge de l'année active + statut
  • Bannière d'avertissement si aucune année active
  • Masquage des accès rapides en mode archive
  • En mode archive, le hook extrait les données du snapshot au lieu d'appeler l'API
