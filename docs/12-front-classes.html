🏫 Gestion des Classes

Pages /classes, /classes/nouvelle, /classes/:id/eleves

Liste des classes — /classes

Grille de ClasseCard paginée (8 par page) + carte dashed « + Nouvelle classe ».

ClasseCard — contenu

  • Header : nom de la classe + badge niveau
  • Meta : icône salle + nom de salle, année scolaire
  • Jauge : « Élèves X / Y » + barre de progression (rouge si ≥ 90%)
  • Actions : Élèves (lien), Planning (lien), ✕ Supprimer (masqué en mode archive)

Créer une classe — /classes/nouvelle

Formulaire avec : nom, niveau (select), année scolaire (3 options dynamiques), capacité (10-50), mode salle (fixe/variable), salle (select depuis SalleContext).

Si aucune salle n'existe, le select est désactivé avec un lien vers /salles.

Élèves d'une classe — /classes/:id/eleves

  • ClasseInfoBar : nb élèves filtrés / total, capacité, salle, champ de recherche
  • ElevesTable : tableau paginé (10/page) avec avatar, genre, date de naissance, âge, email, actions (bulletin + ✕)
