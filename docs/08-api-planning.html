📅 Module Planning

Emplois du temps : créneaux hebdomadaires par classe

Schéma Creneau

{
  classe_id:       String  (requis)  // référence Classe
  matiere_id:      String  (requis)  // référence Matiere
  matiere_nom:     String  (requis)  // dénormalisé pour l'affichage
  matiere_couleur: String  (défaut "#2563eb")
  jour:            Enum    ['Lundi'..'Samedi']
  heure_debut:     String  (requis)  // "08:00"
  heure_fin:       String  (requis)  // "09:00"
  salle:           String  (requis)  // "Salle 101"
  enseignant:      String  (défaut "")
}

Endpoints API

MéthodeRouteDescription
GET/planningTous les créneaux
GET/planning/:idUn créneau
POST/planningCréer un créneau
PATCH/planning/:idModifier
DELETE/planning/:idSupprimer

Grille horaire

Plage : 07:00 → 18:00 par pas de 30 minutes.
Jours : Lundi, Mardi, Mercredi, Jeudi, Vendredi, Samedi.

Dénormalisation

matiere_nom et matiere_couleur sont stockés directement dans le créneau. Cela permet d'afficher le calendrier sans jointure.
